# Walmart vs Target Financial Analysis & Stock Recommendation

## Project Overview

Conducted a comprehensive financial analysis and valuation comparison between Walmart Inc. and Target Corporation using financial statement analysis, ratio analysis, DuPont decomposition, and Discounted Cash Flow (DCF) valuation techniques.

## Objectives

* Compare Walmart and Target financial performance
* Evaluate profitability, efficiency, liquidity, and solvency
* Perform DuPont and RNOA decomposition analysis
* Estimate intrinsic value using DCF valuation
* Provide investment recommendations based on financial analysis

## Financial Analysis Areas

### 1. Profitability Analysis

* Return on Equity (ROE)
* Return on Assets (ROA)
* Return on Net Operating Assets (RNOA)

### 2. Operating Efficiency Analysis

* Accounts Receivable Turnover
* Inventory Turnover
* Accounts Payable Turnover
* PPE Turnover

### 3. Coverage Ratio Analysis

* Times Interest Earned
* Operating Cash Flow to Debt
* Free Cash Flow to Debt

### 4. Liquidity & Solvency Analysis

* Current Ratio
* Quick Ratio
* Debt-to-Equity Ratio
* Liabilities-to-Equity Ratio

### 5. Valuation Analysis

* Discounted Cash Flow (DCF) Valuation
* Sensitivity Analysis
* Intrinsic Value Estimation

* Microsoft Excel

## Skills Demonstrated

* Equity Research

## Key Insights

* Walmart demonstrated stronger operational efficiency and financial stability
* Target showed stronger profitability margins but higher leverage risk
* DCF valuation indicated both companies were fairly valued at market prices
* Walmart provided a stronger risk-adjusted investment profile

## Investment Recommendation

* Walmart (WMT): HOLD
* Target (TGT): HOLD
